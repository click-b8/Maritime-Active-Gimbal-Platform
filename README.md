# Maritime-Active-Gimbal-Platform
This repo serves as reference and overall guide on how to use our Cap Stone ME Senior Design project with everything from high-level architecture and logic to the scripts themselves. This 2-DOF gimbal platform is meant to adapted to any mid-size USV or manned vessel as a drone landing platform capable of maintaning within 2 degrees horizontal over 15 degree disturbances in high sea-states.
